# 开箱即用的夸克网盘资源搜索插件
## 主要功能及示例调用：
- 夸克网盘资源搜索：用户可以通过关键词搜索夸克网盘资源，返回夸克网盘链接
  
  - 示例指令：资源搜索 关键词
    - 返回结果：至多5条资源数据以及分页页码，默认第一页
  
  - 示例指令：资源搜索 -p 2 关键词
    - 返回结果：至多5条资源数据以及分页页码的第二页
